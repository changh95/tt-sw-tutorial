# Tenstorrent 개발자 가이드 (한국어)

Tenstorrent NPU를 처음 쓰는 **PyTorch·GPU 개발자**를 위한 한국어 학습 가이드입니다. 설치(tt-installer)부터 **TT-NN**, **TT-Metalium**, **TT-XLA**, **TT-Lang**, 그리고 **모델 선택(VRAM) · vLLM 배포**까지, 공식 리포지터리·문서의 실제 코드를 그대로 임베드하고 개념 → 실행 → 직접 구축의 학습 경로로 엮었습니다.

전체 페이지는 단일 자기완결형 `index.html` 파일 하나입니다(빌드·프레임워크 없음).

## 보기

- GitHub Pages: 저장소 설정에서 Pages가 활성화되면 제공되는 URL에서 열람할 수 있습니다.
- 로컬: `index.html`을 브라우저로 바로 열면 됩니다.

## 출처

콘텐츠는 Tenstorrent 공식 리포지터리·문서를 한국어로 재구성한 것입니다: `tenstorrent/tt-metal`, `tt-xla`, `tt-lang`, `tt-inference-server`, `tt-installer`, docs.tenstorrent.com. VRAM 계산기는 [cv-learn.com/vram-calculator](https://www.cv-learn.com/vram-calculator/)를 참조합니다.
